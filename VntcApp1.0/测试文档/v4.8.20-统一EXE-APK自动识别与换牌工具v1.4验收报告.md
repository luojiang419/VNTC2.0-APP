## 已知边界

- 工具只支持本项目声明品牌能力且来源可信的母版/本机工具输出，不是任意第三方 EXE/APK 修改器。
- 客户品牌 Android 签名密码由 DPAPI CurrentUser 保护；同名覆盖封装要求同一电脑、同一 Windows 用户。当前版本没有跨用户/跨电脑便携密钥导出，仅复制 profile 无法在其他用户下解密。
- 官方 Android 私钥保存在构建机 `%LOCALAPPDATA%\VNTBrandRepackager\android-official-signing\v1`，不得加入项目、工具或普通源码备份，项目维护者应单独进行离线安全保管。
- Windows 商业代码签名如有需要，应在最终换牌完成后按发布策略执行。
