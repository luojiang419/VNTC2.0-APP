## 使用边界

- 仅接受 v4.8.17 及以后由当前打包脚本生成、带 `VNT_BRAND_READY_V1` 标记的 Windows 品牌母版。
- v4.8.16 及更早的普通安装包没有安全载荷导出协议，工具会在执行前拒绝。
- 品牌包默认关闭官方自动更新，避免升级后恢复官方品牌；后续升级应再次使用同品牌安装包。
- 新图标支持 ICO、PNG、JPG/JPEG、BMP；源文件安全上限为 64 MB、约 4000 万像素。
- 只应加载自己构建或来自可信渠道的品牌母版。
- 当前产物未使用商业代码签名证书，首次在部分电脑运行时可能出现 Windows SmartScreen 提示；这不影响功能验收。
